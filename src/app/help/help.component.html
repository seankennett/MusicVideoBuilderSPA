The frame rate will be 24 frames per second. As the videos are synced up to the right bpm then there is no need for a higher framerate. You can always fiddle with the last ffmpeg command but there are limitations due to the number of files at slower bpms which would result in duplicate frames.

Audio must be compatible with the browser so mp3 is a format that will work more universally.
