Music Video Builder is dedicated to creating videos for music

Why?

We are fed up with seeing static images on youtube and dull backgrounds on dj sets. We want to see better videos for music creators.

Who?

If you have a music file with constant Beats Per Minute then you can use this to create your own video.

How?

Music Video Builder works by providing content and command line arguments to use with the incredible FFmpeg. This way you can purchase a layer and use them in many videos without any extra charges.

There are 3 easy steps to create your video:

First you must have an account with us so you'll need to Sign in using the top right button, we only need a username and email to save your work against.

  1. Use Layer Finder to select layers you're interested in.
  2. Use Clip Builder to combine these layers into short clips.
  3. Use Music Video Builder to combine your clips with your music file to create your video.
